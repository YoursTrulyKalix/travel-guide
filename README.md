# travel-guide
## 🌤️ Best Time to Visit

- **Spring (Mar–May):** Mild weather, fewer crowds  
- **Summer (Jun–Aug):** Hot, busy, great for events  
- **Fall (Sep–Nov):** Cool, scenic, less crowded  
- **Winter (Dec–Feb):** Cold, quiet, good for budget travel  

**Best overall:** Spring or Fall for comfortable weather and fewer crowds.
